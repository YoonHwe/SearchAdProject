[용어 설명]

입찰가: Cost Per Clcik(CPC). 포털 사이트에 광고 컨텐츠를 등록하는 것을 대가로, 컨텐츠가 클릭된 횟수 마다 지불해야 하는 금액. 마케팅 효과 및 효율을 파악하는 데에 있어 중요한 지표.

월간 검색 수: 키워드가 한 달 동안 포털에 검색된 총 횟수.

월간 클릭 수: 등록된 광고 컨텐츠들이 한 달 동안 포털에서 클릭된 총 횟수.

클릭률: Click-Through Rate(CTR). 컨텐츠가 노출되었을 때 해당 컨텐츠가 클릭된 비율로, 마케팅 효과를 파악하는 데에 있어 중요한 지표.

키워드 데이터: 키워드에 대한 수치 데이터. 월간 검색 수, 월간 클릭 수, 클릭률 등을 모두 포함.

노출 수 대비 클릭수: 클릭률, CTR

클릭 당 비용: 입찰가, CPC

키워드 트렌드: 기간 별 키워드 데이터를 파악하여, 변화 추이를 통해 사용자의 키워드 관심도를 파악하는 기능.

월별 검색 수: 각 월 별 검색 수를 나열한 것.

월간 검색 수 비율: 월간 검색 수를 수치 대신 비율로 표현한 것.

Pc 검색 수: 한달 동안 PC에서 해당 키워드가 검색된 총 횟수 

Mobile 검색 수: 한달 동안 Mobile에서 해당 키워드가 검색된 총 횟수 

총 검색 수: Pc 검색 수 + Mobile 검색 수

Pc 클릭 수: 한달 동안 PC에서 해당 키워드에 등록된 광고 컨텐츠가 클릭된 총 횟수 

Mobile 클릭 수: 한달 동안 Mobile에서 해당 키워드에 등록된 광고 컨텐츠가 클릭된 총 횟수

총 클릭 수: Pc 클릭 수 + Mobile 클릭 수

Pc 클릭률: Pc 환경에서만 집계한 클릭률(CTR)

Mobile 클릭률: Mobile 환경에서만 집계한 클릭률(CTR)

평균 클릭률: (Pc 클릭률 + Mobile 클릭률) / 2
